先日、技術書典20に向けて、印刷所「ねこのしっぽ」へCanvaで作成した表紙を入稿しました。Canvaは手軽に使える反面、入稿に向けていくつか注意が必要なポイントがありました。この記事では、実際の作業を通じてわかった注意点をまとめます。「ねこのしっぽ」でCanvaを使うケースに特化した内容ですが、Canvaで入稿素材を作る方の参考になれば幸いです。


##　書籍の入稿について

まず前提として、
- 表紙、背表紙、裏表紙を一枚につなげた画像を作る必要がある
- 無線綴じの場合、背表紙（背幅）の長さは、ページ数、　紙質によって変わる
- そのため、原稿が完成してから、入稿素材を作る必要がある

という点です。

## Canvaでの入稿用データ作成手順

以下は基本的に[ねこのしっぽ Canvaでの原稿制作について](https://www.shippo.co.jp/neko/making/canva.shtml)を参考にしています。ぜひ、こちらもご覧になりながら、Canvaでの入稿用データを作成いただければと思います。

手順の全体像は以下の通りです。

1. ページ数を確定する
2. 背幅を計算する
3. 全体サイズを算出する
4. mmをピクセルに変換する
5. Canvaでカスタムサイズを指定して素材を作る
6. デザインして書き出す


## 事前準備：サイズの確定

入稿用素材は、表表紙・背表紙・裏表紙を1枚の画像にまとめて作成します。背表紙の幅（背幅）はページ数と用紙の厚さによって変わるため、**本文のページ数が確定してから**取り掛かりましょう。

参考：[ねこのしっぽ デジタル入稿ガイド](https://www.shippo.co.jp/neko/making/digital.shtml)

### Step1：背幅を算出するためにページ数を確定する

本文のページ数を確定します。例えば、今回、執筆をした[「dbtとそれにまつわる技術を語ろうじゃないか」](https://techbookfest.org/product/r35huDmDSWBjvqU6uYwENZ)は本文114ページ（表紙を含めると118ページ）でした。

### Step2：背幅を計算する

ねこのしっぽの背幅計算フォームを使い、背幅を計算します。入力項目は以下の通りです。

- 表紙用紙の種類
- 本文用紙の種類
- 総ページ数

今回は最大8mmという結果になりました。なお、背幅が45mmを超えると製本できないため注意が必要です。

![背幅計算フォームを使い、用紙の種類、ページ数をもとに算出](/images/canva-neko-no-shippo-cover-design/book-size.png)


### Step3：全体サイズを算出する

繰り返しになりますが入稿素材は、表表紙・背表紙・裏表紙を1枚の画像にまとめて作成します。そのため、表表紙、裏表紙、背幅をもとにしたサイズを計算します。

今回の作成した書籍はB5サイズになりますので、以下で算出しました。

B5の場合（タチキリ3mm込み）、全体サイズは以下の計算で求められます。

```
幅 = 188mm（表表紙）+ 8mm（背幅） + 188mm（裏表紙）  = 384mm
高さ = 263mm
```

塗り足しが必要な場合は、上下左右にさらに3〜5mm追加します。

### Step4：mmをピクセルに変換する

ここから、Canva固有の事情になるのですが、mmをピクセルに変換する必要があります。理由は、プリセットのサイズを選ぶと解像度が粗くなってしまう上、あとから解像度の変更もできないためです。そこで、自身でピクセルサイズを算出して、入稿素材を作成する必要があります。

算出したmmをピクセルに換算します。ねこのしっぽの場合は、350dpiで算出します。私は、[pixel⇔mm⇔dpi 計算機](https://shimeken.com/print/pixel-mm-dpi)のサイトを利用しました。

最終的に必要なサイズが 5491 x 3634 pixelとわかりました。

### Step5：Canvaでカスタムサイズを指定して素材を作っていく

ここでやっとCanvaの登場です。カスタムサイズを指定して、新規デザインを作っていきます。

![カスタムデザインでピクセルを指定](/images/canva-neko-no-shippo-cover-design/custom-design.png)

また、ガイド機能を使って、画像の中心と背幅をサイズにガイドを引いておくと、作業がしやすいです。

![ガイド機能を使って背幅にガイドを引く](/images/canva-neko-no-shippo-cover-design/set-guide.png)


### Step6：　Canvaで実際にデザインをしていく

最終的なデザインを作成して、入稿用素材を作っていきます。左綴じ、右綴じで表紙の位置が異なりますので、ご注意ください。

![表紙、背表紙、裏表紙を並べていく](/images/canva-neko-no-shippo-cover-design/final-data.png)

最終的にできた素材はねこのしっぽの場合、PNG形式で出力します。データを入稿する際は「Canvaでデータを作成しています、サイズが大きければ調整お願いします」と一言添えるとスムーズです。

### 解像度の修正について

Canvaから書き出したPNG画像は、解像度情報が正しくない状態になっています。

- **書き出し直後**: 解像度 = 96dpi（見た目上のサイズが大きくなる）
- **入稿に必要な状態**: 解像度 = 350dpi

ピクセル数自体は正しいため、画像編集ソフトで解像度を350dpiに変更するだけでOKです。サイズは自動的に正しくなります。

## まとめ

- 入稿素材は表表紙・背表紙・裏表紙を1枚にまとめて作成する
- 背幅はページ数と用紙の種類が確定してから計算する
- キャンバスサイズはプリセットを使わず、カスタムサイズをpxで指定する
- 書き出し後に画像編集ソフトで解像度を350dpiに変更する
- 重要な要素は端から内側に3mm以上の余白を持たせて配置する
- データ入稿時にCanvaで作成した旨を一言添えるとスムーズ

Canva自体は直感的に使えるツールですが、印刷入稿には独自のルールがあります。この記事が同じようにCanvaで表紙を作る方の参考になれば幸いです。
